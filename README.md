# Tridal

Apple Music search + Hi-Res Qobuz streaming for Echo.

## Features

- **Search**: Apple Music catalog (ISRC-based), Tidal fallback
- **Stream**: Qobuz CDN, up to 24-bit/48kHz Hi-Res FLAC
- **Home**: Trending albums, tracks, playlists
- **Albums**: Full track listings
- **Artists**: Top tracks + albums
- **Playlists**: Full track listings

## API

Monochrome HiFi v3.2 — `this-shit-is-not-working.anothermoumen4.workers.dev`

## Build

```bash
./gradlew :app:assembleRelease
```
